## 🚀 Quick Start

<div align="center">

### Installation

</div>

```bash
# Install dependencies
npm install

# Start the application
npm start
```

<details>
<summary>📝 GUI Usage Guide</summary>

1. Enter the target IP/domain
2. Select attack type
3. Set rate limit (default: 100)
4. Click "Start Attack"

</details>
